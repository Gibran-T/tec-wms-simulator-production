# SCN-002 — Preuves attendues (étudiant)

À la fin du scénario, vous devez pouvoir **démontrer** :

1. **GR-2025-001 postée** — la transaction apparaît en statut POST (plus PENDING).
2. **Stock au quai** — SKU-001 visible en REC-01 (quantité > 0).
3. **Conformité rétablie** — aucune alerte « transaction non postée » dans le tour de contrôle.
4. **Pas de double GR** — vous n'avez pas créé une nouvelle réception pour contourner le problème.
5. **Score ≥ 60/100** en mode évaluation.

*Ces preuves correspondent aux critères de réussite. Consultez la Fiche Mission pour le détail des étapes.*
