# SCN-013 — Conditions d'échec

| # | Condition |
|---|-----------|
| 1 | Diagnostic générique sans données |
| 2 | Service analysé sans corréler les erreurs |
| 3 | Plan d'action non mesurable |
| 4 | Ignorer taux d'erreur dans analyse OTIF |
| 5 | Score < 70/100 |

**Anti-patterns :**

- Diagnostiquer service sans erreurs opérationnelles
- Plan générique sans indicateurs
- KPI_DIAGNOSTIC sans chiffres
