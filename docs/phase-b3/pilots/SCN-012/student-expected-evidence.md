# SCN-012 — Preuves attendues (étudiant)

1. **Données KPI consultées** — étape KPI_DATA complétée avant analyse.
2. **Rotation calculée** — formule appliquée avec les chiffres du scénario.
3. **Interprétation claire** — surstock, sous-performance ou bande normale (**4–12×/an**) justifiée par rapport aux données.
4. **Recommandation actionnable** — liée aux données (pas une phrase générique).
5. **Conformité M4** — pipeline KPI terminé.
6. **Score ≥ 70/100** en mode évaluation.
