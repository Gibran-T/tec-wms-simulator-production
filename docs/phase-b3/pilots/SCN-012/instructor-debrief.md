# SCN-012 — Notes débrief instructeur

## Objectif pédagogique

Introduire l'**analyse KPI rotation** comme levier working capital — pas de mouvement physique, focus interprétation.

## Script d'ouverture

> « Vous agissez en tant qu'**Analyste Logistique**.  
> Compétence : **Analyse performance** (Analyser). Seuil **70/100**.  
> Ce scénario est analytique : utilisez le tour de contrôle KPI, pas le tableau transactions. »

## Questions débrief

1. Quelle formule avez-vous utilisée pour la rotation ?
2. Votre rotation est-elle dans la bande 4–12×/an ? Sinon, diagnostic ?
3. Surstock ou sous-performance — quel indicateur corrélaire ?
4. Quelle recommandation opérationnelle (pas seulement descriptive) ?
5. Pourquoi KPI_SERVICE fait partie du pipeline rotation ?

## Signaux d'alerte

- Diagnostic sans chiffres KPI_DATA
- Conclusion surstock sans formule
- Recommandation générique (« améliorer la rotation »)
